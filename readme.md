
1、CardSpace  
   获取卡片的E2P、RAM(DTR/RTR)空间，申请E2P、RAM(DTR/RTR)空间。

2、GPCVM  
   CVM的部分应用。

3、OwnerPin  
   OwnerPin的部分应用。

4、SetATRHistBytes  
  更改ATR历史字节的Applet。

5、TerminateCard  
  卡片锁定，使卡上所有应用功能失效。  

6、GetRandom   
   获取卡片随机数。
   
7、ALG     
   计算MAC、加密。
